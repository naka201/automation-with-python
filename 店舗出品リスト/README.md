### 店舗出品リスト
出品リストのExcelファイルから必要なデータを抽出し、加工しながら、箱番ごとの出品表を作成していくプログラムです。

#### メリット

1. **作業時間の大幅短縮**  
   以前は、出品商品の情報が載っているExcelファイルから手作業で箱番ごとに1商品ずつ必要な情報をコピー＆ペーストし、箱番ごとの出品表を作成していたため、**数時間以上**かかっていました。しかし、このプログラムを使うことで、自動で必要な情報を抽出し、箱番ごとの出品表を作成できるため、作業時間が**数分**に短縮されました。

2. **作業の負担軽減と普及**  
   手作業での出品表作成にはパソコン操作に慣れている社員が必要であり、そのため実施できる社員が限られていました。このプログラムにより、パソコン操作が得意でない社員でも簡単に作業を行えるようになり、作業の**負担が軽減**されました。

3. **エラーのリスク低減**  
   手作業で情報をコピー＆ペーストする際には、**ヒューマンエラー**が発生するリスクがありましたが、プログラムを使用することでそのリスクが排除され、正確で効率的な作業が可能になりました。



#### 必要ファイル
- 出品リスト原本のExcelファイル

#### 注意事項
- プログラムを実行すると、"<年>.<月>"という名前のフォルダが作成され、そのフォルダに各箱番ごとのExcelファイルが作成されて格納されていきます。
- 必要がなくなれば作成されたフォルダは適宜削除してください。
- すべての準備が整ったら、以下のコマンドでプログラムを実行します：  
  ```bash
  python3 店舗出品リスト/main.py
